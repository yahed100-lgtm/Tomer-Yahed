# Catastrophic Forgetting Reconstruction Project (EWC)

============================================
**Description:**
[cite_start]This project implements a full reproduction of Figure 1A from the DeepMind paper: "Overcoming catastrophic forgetting in neural networks" (Kirkpatrick et al., 2017)[cite: 1, 3].
[cite_start]It demonstrates the "Catastrophic Forgetting" phenomenon in standard Neural Networks (SGD) and how the Elastic Weight Consolidation (EWC) algorithm mitigates it[cite: 7].

============================================
**Dataset & Methodology:**
- **Dataset:** Permuted MNIST. Task A is the original MNIST dataset. Tasks B and C are generated by applying a fixed random permutation to the image pixels.
- **Algorithm:** The EWC algorithm calculates the Fisher Information Matrix after each task to identify critical weights. A penalty is added to the loss function to protect these weights during subsequent tasks.

============================================
**Requirements:**
- Python 3.x
- torch, torchvision
- matplotlib, numpy

============================================
**Execution Instructions:**
1. Run the main script: `python main.py`
2. The script will automatically:
   - Generate the 3 sequential tasks (Permuted MNIST).
   - Train a standard MLP (SGD Baseline) for 10 epochs per task to show catastrophic forgetting.
   - Train an identical MLP with EWC protection to demonstrate knowledge retention.
   - Save the comparison graph to the project directory.

============================================
**Output:**
- `figure_1A_publication_ready.png`: A graph replicating the original paper's results. It shows the SGD baseline accuracy on Task A dropping as new tasks are learned, while the EWC model maintains high accuracy.

============================================
**AI Collaboration & Logs (תיעוד עבודה מול AI):**
As per the project requirements, a traditional summary report is replaced with full AI interaction documentation.
- **Logs Location:** All transcripts and problem-solving processes with the AI are saved in the `AI_Logs/` directory in this repository.

#  EWC פרויקט שחזור: התגברות על שכחה קטסטרופלית בעזרת 

============================================
**תיאור הפרויקט:**
פרויקט זה מציג שחזור מלא ומדויק של איור 1A (Figure 1A) מתוך המאמר המדעי של חברת DeepMind: 
"Overcoming catastrophic forgetting in neural networks" (Kirkpatrick et al., 2017).
הפרויקט מדגים את תופעת "השכחה הקטסטרופלית" ברשתות נוירונים סטנדרטיות (SGD) כאשר הן לומדות משימות ברצף, ומראה כיצד אלגוריתם ה-Elastic Weight Consolidation (EWC) פותר בעיה זו ומשמר ידע קודם.

============================================
**מערך הנתונים ושיטת העבודה:**
- **מערך הנתונים (Permuted MNIST):** כדי לדמות למידה של משימות חדשות ללא שינוי ארכיטקטורת הרשת, השתמשנו בשיטת Permuted MNIST (כפי שבוצע במאמר). משימה A היא ה-MNIST המקורי. משימות B ו-C נוצרות על ידי ערבוב אקראי וקבוע של פיקסלי התמונות.
- **האלגוריתם (EWC):** לאחר למידת כל משימה, האלגוריתם מחשב את מטריצת המידע של פישר (Fisher Information Matrix) כדי לזהות אילו משקולות ברשת היו קריטיות להצלחת המשימה. בלמידת המשימות הבאות, מתווסף "קנס" (Penalty) לפונקציית ההפסד שמונע שינויים דרסטיים במשקולות החשובות הללו.

============================================
**דרישות מערכת:**
- Python 3.x
- ספריות: `torch`, `torchvision`, `matplotlib`, `numpy`

============================================
**הוראות הרצה:**
1. הרץ את הקובץ הראשי: `python main.py`
2. הסקריפט יבצע אוטומטית את השלבים הבאים:
   - ייצור 3 משימות עוקבות (Permuted MNIST).
   - יאמן רשת סטנדרטית (SGD Baseline) למשך 10 איטרציות לכל משימה כדי להדגים שכחה קטסטרופלית.
   - יאמן רשת זהה לחלוטין עם מנגנון הגנה EWC כדי להדגים שימור זיכרון.
   - יפיק וישמור את גרף ההשוואה.

============================================
**פלט התוכנית:**
- הסקריפט מייצר את התמונה `figure_1A_publication_ready.png`.
